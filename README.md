## Key Mathematical Algorithms & Concepts Used
Some important algorithms and theorems implemented include:

- **Euclidean Algorithm**: Efficient method to compute GCD using repeated remainder operations.
- **Extended Euclidean Algorithm**: Finds coefficients satisfying `ax + by = gcd(a,b)`.
- **Binary Exponentiation**: Fast exponent calculation using repeated squaring.
- **Sieve of Eratosthenes**: Technique to generate prime numbers efficiently.
- **Euler’s Totient Function (φ(n))**: Counts integers coprime with n.
- **Chinese Remainder Theorem**: Solves simultaneous modular congruences.
- **Taylor Series Expansion**: Approximates functions like sin(x), eˣ, and ln(x).
- **Monte Carlo Method**: Uses random sampling for numerical approximation.
- **Newton–Raphson Method**: Iterative root-finding technique.
- **Simpson’s and Trapezoidal Rules**: Numerical integration methods.
- **Romberg Integration**: Improves integration accuracy using extrapolation.
